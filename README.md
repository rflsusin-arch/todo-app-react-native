# 📱 To-Do List App — React Native

Aplicativo mobile de lista de tarefas desenvolvido com React Native e Expo.

## Funcionalidades

- Exibição da data atual em português
- Contagem dinâmica de tarefas pendentes e realizadas
- Listagem de tarefas separadas em duas seções: Pendentes e Realizadas
- Cada tarefa exibe nome, emoji e categoria
- Checkbox para marcar/desmarcar tarefas
- Botão flutuante (FAB) para adicionar nova tarefa
- Modal com animação slide para criação de tarefas
- Seletor de emojis inline
- Fechar modal ao tocar fora

## Tecnologias utilizadas

- React Native
- Expo
- expo-checkbox
- expo-status-bar
- @expo/vector-icons
- react-native-emoji-chooser
- @react-native-async-storage/async-storage

## Como executar

1. Clone o repositório
2. Instale as dependências: `npm install`
3. Inicie o projeto: `npx expo start`
4. Escaneie o QR Code com o app Expo Go
